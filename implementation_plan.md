# 더프라임다온 공식 웹사이트 구축 구현 계획

'더프라임다온'은 B2B 중심의 가죽 제품 제조 전문 업체로, 신뢰도 높은 기술력과 철저한 납기/품질 관리를 강조하는 공식 웹사이트를 구축하고자 합니다. 본 계획서는 이를 위한 아키텍처와 디렉토리 구조, 그리고 초기 개발 방향을 제안합니다.

---

## 1. 아키텍처 및 기술 스택 제안

B2B 웹사이트는 잠재 고객사(기업 구매 담당자)가 포털 검색을 통해 유입될 확률이 매우 높으므로 **SEO(검색엔진 최적화)**와 **빠른 페이지 로드 속도**가 최우선입니다.

- **Frontend**: HTML5, Vanilla CSS (최신 모던 CSS 스타일링), Vanilla JavaScript
  - 빌드 도구나 무거운 프레임워크 없이도 초고속 로딩이 가능하며, 검색엔진 로봇이 콘텐츠를 완벽하게 인덱싱할 수 있습니다.
  - 유지보수가 직관적이고 호스팅(Netlify, GitHub Pages, 일반 웹호스팅 등)이 자유롭습니다.
- **디자인 라이브러리 (선택적)**: FontAwesome (아이콘), Google Fonts (Outfit 및 Noto Sans KR)
- **주요 전략**: B2B 특화 상담 문의(CTA) 버튼을 상시 노출하고, 상세한 폼을 구성하여 전환율을 극대화합니다.

---

## 2. 사이트맵 및 페이지별 기획

웹사이트는 검색 키워드 타겟팅과 구조적 신뢰감을 주기 위해 **멀티 페이지(Multi-Page) 구조**로 제안합니다.

| 페이지 명 | 주요 내용 | SEO 타겟 키워드 |
| :--- | :--- | :--- |
| **메인 (`index.html`)** | 핵심 제조 전문성, 기술력/품질 요약, 최신 쇼룸 프리뷰, 주요 파트너사, 문의 CTA | 가죽제품 OEM, 가죽제품 ODM, 가죽 제조 공장 |
| **회사 소개 (`about.html`)** | 경영 철학, 대표 인사말, 연혁, 파트너사 레퍼런스, 오시는 길 | 더프라임다온, 가죽 판촉물 제작 업체 |
| **제조 역량 (`capabilities.html`)** | 보유 설비(재단기, 피할기, 불박기 등) 리스트, Making Process(기획-개발-생산-검수-납기) 단계별 소개 | 가죽 임가공, 가죽제품 대량생산, 가죽 불박 인쇄 |
| **제품 쇼룸 (`showroom.html`)** | 주요 포트폴리오(폰케이스, 키케이스, 워치스트랩, 라이프스타일 굿즈 등), 카테고리 필터링 | 가죽 폰케이스 도매, 가죽 키케이스 제작, 가죽 스트랩 OEM |
| **상담 문의 (`contact.html`)** | B2B 특화 견적/샘플 신청 폼 (기업명, 담당자명, 연락처, 품목, 예상 수량, 첨부파일 등) | 가죽 제품 제작 문의, 가죽 OEM 견적 |

## 3. 디자인 톤앤매너

- **컬러 팔레트**:
  - **Primary**: Deep Navy (`#0F172A`) - 신뢰, 전문성, 정교함
  - **Secondary / Accent**: Warm Taupe / Gold Brown (`#854D0E`) - 가죽 원단의 클래식함과 고급스러움 표현
  - **Neutral**: Slate Gray (`#64748B`, `#F8FAFC`) - 가독성 높은 배경 및 본문 색상
  - **Base**: White (`#FFFFFF`)
- **타이포그래피**:
  - 영문: `Outfit` (세련되고 모던한 지오메트릭 산세리프)
  - 국문: `Noto Sans KR` (본문 가독성이 우수한 기본 서체)
- **레이아웃**: 와이드하고 여백을 충분히 둔 미니멀 레이아웃, 스무스한 마이크로 인터랙션(Hover 효과, 스크롤 애니메이션) 적용
